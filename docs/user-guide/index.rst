##################
NETCONF User Guide
##################

This guide contains information on how to use OpenDaylight's NETCONF plugin. Users must have
basic knowledge of the concepts, supported capabilities, configuration, and usage of
NETCONF. For more information on NETCONF, refer to `RFC 6241 <https://tools.ietf.org/html/rfc6241>`_.
